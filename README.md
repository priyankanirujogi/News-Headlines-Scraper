# News Headline Scraper

## Objective
Scrape top news headlines from BBC News and save them into a text file.

## Technologies Used
- Python
- Requests
- BeautifulSoup

## Files
- news_scraper.py
- headlines.txt

## How to Run

Install dependencies:

pip install requests
pip install beautifulsoup4

Run:

python news_scraper.py

## Output
The headlines are saved in headlines.txt
